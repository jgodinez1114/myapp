# myapp
This app  uses Express JS and starts a server and listens on port 3000 for connections. The app responds with “Hello World!” for requests to the root URL (/) or route. For every other path, it will respond with a 404 Not Found.
## Express JS
### Running the app locally
* Per the Express docs, 
> "First create a directory named myapp, change to it and run npm init."
